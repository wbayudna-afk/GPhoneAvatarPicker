# Gartic Phone Avatar Picker

A lightweight Chrome extension (unpacked) that allows you to easily pick and customize your avatar in [Gartic Phone](https://garticphone.com).

## Features

- Browse and select avatars from a clean interface
- Quick avatar switching without page reloads
- Works directly within the Gartic Phone game UI

## Installation (Unpacked Extension)

Since this extension is not published on the Chrome Web Store, you need to load it manually in developer mode.
